# Informe Diario Regional

Aplicación de Streamlit para la generación de informes regionales utilizando Groq y procesamiento de feeds.

## Requisitos

- Python 3.8+
- Streamlit
- Groq API Key

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/SebasQuintero17/informe-diario-regional.git
   ```
2. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```
3. Ejecuta la aplicación:
   ```bash
   streamlit run app.py
   ```
